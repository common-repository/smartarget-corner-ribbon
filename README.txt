== Description ==

Corner Ribbon - Promote special offers with Corner Ribbon
Do you have special deals sometimes? Like Black Friday? Weekend special? Holidays?
The Corner Ribbon will be shown to users on all pages, both on desktop and mobile. It allows you to communicate with them about your promotion.
Smartarget Corner Ribbon plugin works also with WooCommerce, Elementor and Oxygen Builder.

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
